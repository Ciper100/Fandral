About Fandral Editor

Fandral editor is a minimal and beginner friendly text editor focused on improving text editing speed for general users. The improvement in text editing speed is achieved by the help of command environment as available in text editors like vim and emcacs. Since, command can be hard and may have steep learning curve, all the functionality is available through GUI. You can get familiar with the command while using the text editor as any other GUI text editors like notepad++ or gedit. For exact information on how to use command line, you can use help option.
