Multi Character Commands

  • : => Directs the command mode into MultiChar command mode
  • :wq => Saves the current file (performs the same operation as through the GUI save button or "Ctrl + S" have been pressed) and quits the tab.
  • :q => Quit the tab, asks to save if the file isn't saved previously
  • :w => Save the current file
  • :q! => quits the tab without asking the user to save (throwing away the changes)
