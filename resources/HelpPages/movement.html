Movement

  • k : Jumps the text cursor one line up until the first line is reached, pressing it continuously moves the cursor up until the first line is reached.
  • j : Jumps the text cursor one line down, pressing it continuously will move the cursor down until the last line is reached.
  • h : Jumps the text cursor one character towards left, when a new line is encountered it jumps one line above(if available),pressing it continuously will move the cursor towards the start of the document.
  • l : Jumps the text cursor by one character towards right, when a new line is encountered it jumps one line below (if available), pressing it continuously will move the cursor towards right until the end of the document is found.
