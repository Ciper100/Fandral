Copy

If the editor is in insertion mode, then you can copy and paste normally
i.e. by using "Ctrl + C " and "Ctrl + V" But if you're in command mode, you can press "yy" to copy the current line.

Copying a single word isn't support yet and will be added soon.

Cut

If the editor is in insertion mode, then you can cut and paste normally
i.e. by using "Ctrl + X " and "Ctrl + V"
But if you're in command mode, you can press "dd" to cut the current line.
Cutting a single word isn't support yet and will be added soon.
You can also press x to cut the character right of the cursor.

Paste

If the editor is in insertion mode, then you can paste normally
i.e. by using "Ctrl + V"
But if you're in command mode, you can press p or P to paste the current clipboard content.
